# agent-desktop（Next.js）

这是一个基于 Next.js App Router 的前端应用，用于承载客服 Copilot 的界面能力（如客服工作台、知识库后台）。

## 本地开发

```bash
pnpm dev
```

默认访问：`http://localhost:3000`

## 常用命令

```bash
# 开发
pnpm dev

# 构建
pnpm build

# 生产启动
pnpm start
```

## 代码入口

- `app/page.tsx`：首页入口
- `app/layout.tsx`：全局布局
- `app/globals.css`：全局样式
