# 客服 Copilot API 调用示例

## 启动服务
```bash
pnpm --filter agent-service start:dev
```

## 健康检查
```bash
curl http://localhost:3006/health
```

## 新增知识库文章
```bash
curl -X POST http://localhost:3006/api/knowledge/articles \
  -H 'Content-Type: application/json' \
  -d '{
    "title": "会员续费规则",
    "content": "会员到期后可在30天内补差价续费。",
    "tags": ["会员", "续费"]
  }'
```

## 生成回复草稿
```bash
curl -X POST http://localhost:3006/api/copilot/draft-reply \
  -H 'Content-Type: application/json' \
  -d '{
    "conversationId": "conv-001",
    "customerId": "cust-001",
    "customerQuestion": "我订单一直没发货，可以退吗？",
    "channel": "wechat"
  }'
```

## 创建升级工单
```bash
curl -X POST http://localhost:3006/api/tickets \
  -H 'Content-Type: application/json' \
  -d '{
    "conversationId": "conv-001",
    "customerId": "cust-001",
    "title": "二线介入：复杂售后问题",
    "description": "客户要求加急退货并投诉物流延迟",
    "priority": "high"
  }'
```

## 查询审计日志
```bash
curl "http://localhost:3006/api/audit/events?limit=20"
```
