# 客服 Copilot API 调用示例

## 一次性准备（首次）
```bash
cd apps/agent-service

# 1) 启动本地数据库（pgvector）
pnpm run db:up

# 2) 执行 migration
pnpm run db:run

# 3) 启动服务
pnpm run start:dev
```

## 常见错误：ECONNREFUSED 127.0.0.1:5432
表示数据库没连上，依次检查：
1. 容器状态：`pnpm run db:logs`
2. 用户密码是否一致（默认 root / zjf012511）
3. 5432 端口是否冲突

## 健康检查
```bash
curl http://localhost:3006/health
```

## 新增知识库文章
```bash
curl -X POST http://localhost:3006/api/knowledge/articles \
  -H 'Content-Type: application/json' \
  -d '{
    "title": "会员续费规则",
    "content": "会员到期后可在30天内补差价续费。",
    "tags": ["会员", "续费"]
  }'
```

## 生成回复草稿
```bash
curl -X POST http://localhost:3006/api/copilot/draft-reply \
  -H 'Content-Type: application/json' \
  -d '{
    "conversationId": "conv-001",
    "customerId": "cust-001",
    "question": "我订单一直没发货，可以退吗？",
    "channel": "wechat"
  }'
```

## 创建升级工单
```bash
curl -X POST http://localhost:3006/api/tickets \
  -H 'Content-Type: application/json' \
  -d '{
    "conversationId": "conv-001",
    "customerId": "cust-001",
    "title": "二线介入：复杂售后问题",
    "description": "客户要求加急退货并投诉物流延迟",
    "priority": "high"
  }'
```

## 查询审计日志
```bash
curl "http://localhost:3006/api/audit/events?limit=20"
```
